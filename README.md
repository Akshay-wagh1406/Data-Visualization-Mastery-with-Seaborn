# Interactive Sales Dashboard

## Project Description

This project creates an Interactive Sales Dashboard using Python libraries like Pandas, Seaborn, Matplotlib, and Plotly.

The dashboard helps analyze:

- Sales trends
- Product performance
- Customer segmentation
- Profit analysis

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook

---

## Features

- Statistical plots using Seaborn
- Interactive charts using Plotly
- Heatmaps and correlations
- Customer segmentation
- Multi-plot dashboards
- Professional dashboard layout

---

## Installation


pip install -r requirements.txt

---

## Run Project

python dashboard.py


## Output

- Sales Trend Analysis
- Interactive Dashboard
- Heatmaps
- Customer Analysis
- Product Performance Reports
